# 系统测试计划

## 测试环境

| 项目 | 配置 |
|------|------|
| **后端** | 本地 `http://localhost:8000` |
| **前端** | 本地 `http://localhost:7500` |
| **模型** | h3i 平台（需关闭代理） |
| **数据库** | SQLite（本地 `writing_assistant.db`） |
| **浏览器** | Chrome / Edge 最新版 |

> 注意：测试前确保后端已启动且健康检查通过（`curl http://localhost:8000/api/health`）

## 测试用例

### 1. 会话管理

| 编号 | 测试项 | 操作步骤 | 预期结果 |
|------|--------|---------|---------|
| 1.1 | 新建会话 | 点击"新建会话" | 会话列表新增一项，切换到开始页面 |
| 1.2 | 切换会话 | 点击列表中的其他会话 | 加载该会话的内容和聊天历史 |
| 1.3 | 重命名会话 | 悬停会话，点击重命名，输入新名称 | 会话名称更新 |
| 1.4 | 删除会话 | 点击删除按钮 | 会话从列表移除，切到其他会话或空状态 |
| 1.5 | 刷新页面 | F5 刷新 | 记住当前会话，有内容时自动进入内容页 |
| 1.6 | 创建无会话时 | 删除所有会话 | 显示"请创建会话" |

### 2. 快速写作

| 编号 | 测试项 | 操作步骤 | 预期结果 |
|------|--------|---------|---------|
| 2.1 | 通用公文 | 选择"快速写作"→"通用公文"→输入要求→"开始生成" | 流式生成，内容逐字出现，生成完成后切换到编辑器 |
| 2.2 | 通知 | 选择"通知"→输入要求→生成 | 生成符合通知格式的公文 |
| 2.3 | 规章制度 | 选择"规章制度"→输入要求→生成 | 生成章条款结构的制度 |
| 2.4 | 讲话稿 | 选择"讲话稿"→输入要求→生成 | 生成有开场问候的讲话稿 |
| 2.5 | 空要求 | 不输入要求直接点生成 | 按钮禁用，无法生成 |
| 2.6 | 生成耗时 | 记录从点击到首 chunk 的时间（看 console.time） | TTFB 应在 3-8 秒内（含 RAG 检索时间） |

### 3. 参考写作

| 编号 | 测试项 | 操作步骤 | 预期结果 |
|------|--------|---------|---------|
| 3.1 | 上传参考文档 | 选择"参考写作"→上传 .md 文件→选择"基于内容生成"→补充要求→生成 | 文件上传成功，生成基于参考内容的文章 |
| 3.2 | 上传 PDF | 上传 .pdf 文件→生成 | 文件上传并解析成功 |
| 3.3 | 仿写公文 | 上传文档→选择"仿写公文"→输入新主题 | 生成风格类似但内容不同的新公文 |
| 3.4 | 生成回函 | 上传文档→选择"生成回函"→生成 | 生成正式回函（复函格式） |
| 3.5 | 无上传时 | 不传文件直接点生成 | 按钮禁用 |

### 4. 编辑器功能

| 编号 | 测试项 | 操作步骤 | 预期结果 |
|------|--------|---------|---------|
| 4.1 | 目录导航 | 点击目录中的标题 | 编辑器滚动到对应位置 |
| 4.2 | 编辑模式 | 点击"编辑"→修改内容→"保存" | 内容保存成功 |
| 4.3 | 取消编辑 | 编辑→"取消" | 内容恢复为保存前的版本 |
| 4.4 | 内容审查 | 点击"内容审查" | 显示审查结果（A100 上有效，本地可能失败） |
| 4.5 | 字符统计 | 查看底部状态栏 | 显示正确的字符数、行数、标题数 |

### 5. 修改润色

| 编号 | 测试项 | 操作步骤 | 预期结果 |
|------|--------|---------|---------|
| 5.1 | 全文修改 | 在聊天输入框输入修改要求→发送 | 编辑器内容实时更新（逐字刷新） |
| 5.2 | 选中文本修改 | 在编辑器选中一段文本→弹出 AI 对话框→输入要求→确认修改 | 编辑器内容增量更新 |
| 5.3 | 空文章时 | 没生成内容时在聊天框发修改 | 提示"请先生成文章内容" |
| 5.4 | 聊天历史 | 修改完成后查看聊天区域 | 显示用户的修改要求和 AI 的修改说明 |

### 6. 模板管理

| 编号 | 测试项 | 操作步骤 | 预期结果 |
|------|--------|---------|---------|
| 6.1 | 开始界面管理 | 点击开始界面底部"管理模板"→展开面板 | 显示当前模板列表 |

## 测试完成标准

1. 所有功能测试用例全部通过
2. 未发现导致流程中断的 Bug
3. 响应时间在可接受范围内（TTFB < 10s）
4. 数据持久化正常（刷新后不丢失）
